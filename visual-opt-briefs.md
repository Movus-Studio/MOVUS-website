# MOVUS Visual System — Photo Briefs (visual-opt branch)

## Setup in Higgsfield UI for every gen

- **Model:** Z Image (`z_image`) — validated 2026-05-26, single-kettlebell deadlift came back 9/10
- **Aspect ratio:** 4:5 portrait
- **Quality:** base (free on unlimited tier)
- **Negative prompt:** LEAVE EMPTY
- **Soul-only UI features (Moodboard, Hex):** N/A on Z Image. The inline hex codes in each prompt drive the color directly.

## Hard-won rules baked into these prompts

- **Multi-figure is unlocked on Z Image.** Soul V2 collapsed to magazine pyramid every time; Z Image renders the silhouette + gradient cleanly so we can bring back group + coach compositions where they earn their place.
- **No barbells.** Use bodyweight or single simple equipment (kettlebell, dumbbell, plyo box, sled, jump rope). Barbells render with floating bars / single-end plates.
- **Real gym poses only.** No mid-air jumps unless plyometric and grounded. No ballet stretches. No "mid-stretch" unless it's the deliberate wellness slot. Poses sourced from authentic strength-and-conditioning movement library.
- **Silhouette / heavy shadow** is the visual register. Face obscured by shadow, angle, or gesture. Form over identity.
- **Founder Soul ID dropped from this set.** Silhouettes don't show faces, so the founder-as-coach trust signal moves to a future lit-register shoot.
